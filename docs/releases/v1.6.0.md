# HLS Downloader v1.6.0

## 本机文件推送到电视

- 桌面端现在可以选择任意本机媒体文件，或从已完成下载的详情中直接推送到 TVBox。
- 软件只为被选中的单个文件创建带随机令牌的临时只读局域网链接，支持 HTTP Range 请求；不会公开下载器 API 或整个文件夹。
- 推送前会二次确认，明确显示共享与自动清理规则。
- 文件传输结束、电视停止访问约 2 分钟后会自动撤销；最长保留 2 小时。状态栏会检测撤销结果，可随时“停止共享”；程序退出也会一并清理。
- 文件格式不在选择器中被限制；实际能否播放取决于电视端对容器和编解码器的支持。

## 界面工作台

- 主界面改为一体化桌面工作台：固定左侧导航、顶部操作栏、中央任务表和底部状态栏。
- 移除悬浮卡片式外壳和过多阴影，保留高密度下载信息与清晰层级。
- 浏览器插件的“推电视”同样增加二次确认，不增加额外权限。
