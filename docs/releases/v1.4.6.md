# HLS Downloader v1.4.6

## 功能

- **状态栏限速快捷档**：不限速 / 256–10 MiB/s，即时写入设置并应用到下载令牌桶。
- **启动时应用限速**：服务启动即 `configure`，重启后仍生效。
- **全部开始 / 全部暂停**：工具栏一键处理排队、暂停与可暂停的活动任务。
- **失败筛选**：侧栏「失败」聚合 failed + unsupported。
- **复制下载链接**：任务右键菜单一键复制 URL。
- **批量添加对话框**统一到 Dialog/Field/Input 原语，文案支持多协议链接。
