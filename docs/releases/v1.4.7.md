# HLS Downloader v1.4.7

## 视频识别大幅降噪

这是本版核心：以前识别视频会刷出一大堆噪声链接，现在做了三层收敛。

### 浏览器插件

- HLS 主清单的多个码率变体折叠到一条资源下，不再每条变体单独刷屏；面板内可直接选清晰度（自动最高 / 1080p / 720p …）。
- 失败响应（4xx/5xx）和非 GET/OPTIONS 请求不再进入资源列表。
- 分片 MIME（video/mp2t、audio/aac）和 init/segment/chunk URL 被过滤。
- 广告信号 URL 在缺少时长/大小时排除。
- 带签名参数的刷新 URL 稳定去重（token、expires、X-Amz-Signature 等不参与 fingerprint），真正有意义的 quality 参数保留。
- 资源按可信度排序：主清单 > 单条 HLS > 媒体文件 > 普通文件；有 variants、有时长、有分辨率额外加分。
- 面板和 popup 最多展示 8 条视频资源，文件噪声在有视频时隐藏。

### 桌面端手工识别

- 后端候选提取重写：HTML 解析更严格，排除 placeholder、字幕/缩略图/广告路径，候选上限从 100 降到 12。
- 每个候选带 label（文件名）、quality（推测清晰度）、confidence（可信度）。
- 前端候选列表从原始长 URL 按钮改为结构化卡片：文件名、域名、清晰度标签、来源、推荐标记；默认推荐最佳项，回车直接下载。
