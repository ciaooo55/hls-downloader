# HLS Downloader v1.5.0

## TVBox 局域网推送

- 桌面端新增局域网 TVBox 扫描：仅扫描本机私有 IPv4 网段和常见 TVBox 端口（9978/9979/9977/9976）。
- 扫描结果可在设置中选择，保存后会记住设备地址；也支持手动填写地址。
- 插件只保留“推电视”入口，通过 Native Messaging 将当前视频地址交给桌面端，不保存 TVBox 地址，也不直接访问局域网设备。
- 推送失败会明确提示未配置设备、地址错误或电视端拒绝，不会误报成功。

## 兼容性与审核友好

- 保持 Tauri 2 + React 19 桌面壳；安装升级继续清理旧 Compose 壳残留。
- 插件不新增 TVBox 专用网络权限；局域网扫描和配置全部由桌面软件完成。
- 配置版本升级到 11，旧配置会自动补齐 `tvbox_endpoint`，已有下载设置不受影响。
