# HLS Downloader v1.6.3

## 紧凑窗口也完整可用

- 主窗口最低尺寸调整为 **820 × 560**；不最大化时会按空间自动收敛，而不是裁切工具栏或任务内容。
- 窄窗口会将侧栏收为带悬停提示的图标轨，并只保留任务表的文件、状态、进度和操作列。
- 次要工具会进入“更多操作”菜单；批量添加、全局开始/暂停、恢复、打开文件、日志、浏览器插件、本机媒体推送和更新检查仍可访问。
- 底部状态栏和纵向间距同步压缩，确保下载的核心流程在低高度窗口仍然清晰可操作。

## 视频与 BT 可靠性

- 无扩展名但带 HLS/DASH MIME 的浏览器资源会正确创建自适应媒体任务，不再被误当成普通文件下载。
- 播放器浮动按钮只会对与当前 `<video>` / `<source>` 精确关联的唯一资源一键下载；blob/MSE 或不确定候选必须先由用户选择。
- 插件进一步隐藏音轨、字幕、广告、预览、缩略图和 storyboard HLS/DASH 清单，减少噪声。
- BT 下载中可以实时调整要下载的文件；已取消文件会停止请求，已下载的数据保留。
