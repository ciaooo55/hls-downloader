# HLS Downloader v1.4.8

## 浏览器插件：推送到电视（TVBox）

在插件资源面板和 popup 里新增「推电视」按钮，可将当前页面的视频资源直接推送到局域网内的 TVBox/影视盒子播放。

### 使用方式

1. 在插件 popup 顶部点击「设置电视」，输入 TVBox 推送地址（例如 `http://192.168.1.100:9979`）。
2. 在视频页面的悬浮面板里，每条资源旁有「下载」和「推电视」两个按钮。
3. 点击「推电视」即可将资源 URL 推送到电视端播放。

### 技术细节

- 使用 TVBox 标准 push 协议：`POST {endpoint}/action`，body 为 `do=push&url=...`。
- 对不支持 POST 的 fork 自动回退到 GET 方式。
- 推送前校验地址格式（必须 http/https），未设置时提示先配置。
- 地址存储在 `browser.storage.local`，支持跨会话保持。
- 页内悬浮面板和 popup 均可推送，不依赖桌面端在线。
