# HLS Downloader v1.4.5

## 功能

- **全局下载限速**：设置 → 任务调度可配置 KiB/s（0 不限速）；HTTP/HLS 分片共享令牌桶预算。
- **新建重复确认**：手工/识别添加相同 URL 时返回 409，界面二次确认后可强制添加（`allow_duplicate`）。
- **对话框原语迁移**：新建下载、浏览器接管、插件安装框统一使用 Button/Dialog/Input/Field。
- **API 错误结构化**：重复链接返回 `{ code, message, duplicates }`，前端可解析并提示。

## 验证

- Python 全量回归（含 throttle / duplicate）。
- 前端 vitest + 生产构建。
- Windows Release 工作流产物。
