# HLS Downloader v1.4.3

## 架构与界面

- 明确锁定主流产品栈：Tauri 2 + React 19 + Vite 7 + Tailwind CSS v4 + Zustand + cva/clsx（见 `docs/architecture.md`、`DESIGN.md`）。
- 新增 UI 原语：`Button` / `Dialog` / `Badge` / `Input` / `Field`，确认框与日志窗已迁到统一 Dialog。
- Cockpit 工作台 token 调整为更密的产品工具风格（冷灰 slate + 单蓝主色，减少毛玻璃装饰）。

## IDM 级队列

- 排队任务支持优先级调度：同时任务数内按优先级启动，而不再是简单的先到先得信号量。
- 任务右键菜单新增：队列上移 / 下移 / 队首 / 队尾（`POST /api/tasks/{id}/queue/{direction}`）。
- 任务列表在排队状态显示队列位次。

## 403 诊断

- 细分 Cloudflare 拦截（`HTTP_403_CLOUDFLARE`）与签名链接过期（`HTTP_403_EXPIRED_SIGNATURE`）。
- 失败详情增加可执行建议步骤（刷新页面、扩展重发、授权 Cookie、完成人机验证等）。
