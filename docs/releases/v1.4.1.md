# HLS Downloader v1.4.1

## 修复内容

- 桌面安装版和便携版现在实际使用 Tauri + React 主程序，不再把旧 Kotlin/Compose 主界面打入正式包。
- 主界面采用 Cockpit Tools 同类的悬浮侧边栏、独立工具栏与工作区布局；深色、浅色、焦点和减少动态效果均保持一致。
- “边下边播”和“内置播放”在同一个桌面窗口内打开本地播放器，不会跳转到 `127.0.0.1` 的网页界面；仍保留播放进度、缓冲、缩略图、音量、倍速、画中画和全屏。
- 修复 FastAPI 返回结构化 422 校验错误时，旧客户端将 JSON 数组强制解析为字符串而崩溃的问题；现在会显示实际字段错误。
- 完善中文字体回退链，避免系统界面缺少首选字体时显示方块。
- Chrome 和 Firefox 插件版本同步升级至 1.4.1。

## 验证

- React 前端测试、Chrome/Firefox 插件测试与两个生产构建。
- Tauri release 构建、Windows 安装版/便携版打包、启动健康检查、单实例、优雅退出与 Native Messaging 冒烟测试。
- 发行文件均包含在 `SHA256SUMS.txt`。
