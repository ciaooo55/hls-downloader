# HLS Downloader v1.4.9

## TVBox 推送稳定性细节

- 将 TVBox 协议抽成可测试模块，统一处理 endpoint、`/action` 路径和 URL 编码。
- 用户输入地址即使带 `/action` 或尾部斜杠，也不会拼出重复路径。
- 识别 TVBox 返回的 JSON/文本错误，不再把 HTTP 200 的失败响应误判为成功。
- popup 和页内面板共用同一套协议处理。
- 推送地址为空、格式错误、非 HTTP(S) 和网络不可达时，显示明确中文提示。
