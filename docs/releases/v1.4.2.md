# HLS Downloader v1.4.2

## 修复与改进

- 安装器升级时会删除 v1.4.0 Kotlin/Compose 遗留的 `app` 与 `runtime` 目录，防止旧桌面壳和新下载核心混用；维护页会显示实际桌面壳为 Tauri + React 及其版本。
- 新建任务可选填 MD5、SHA-1 或 SHA-256。HTTP、HLS、DASH 和单文件 BT 都会在写入最终文件后、标记完成前校验。
- 校验失败时任务明确标记为失败，保留最终文件和实际摘要供核对；多文件 BT 不会伪造单一校验结果。
- 浏览器连接说明只保留正式浏览器扩展，不再提及已移除的油猴脚本。

## 验证

- Python 下载、API、安装器规则与校验和回归测试。
- React 类型检查、前端测试和生产构建。
- Windows 安装版、便携版、Chrome/Firefox 扩展、Firefox lint 和 SHA-256 发行文件由 Release 工作流构建并验证。
